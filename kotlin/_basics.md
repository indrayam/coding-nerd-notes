# Kotlin Basics

## Types
- Integers
  - Types:
    - `Byte` (8 bits)
    - `Short` (16 bits)
    - `Int` (32 bits)
    - `Long` (64 bits)
  - Integer type defaults to type `Int`
  - Binary number defined with `0b` prefix
  - Hexadecimal number defined with `0x` prefix
- Floating Points
  - Types:
    - `Float` (32 bits)
    - `Double` (64 bits)
  - Floating points defaults to type `Double`
- Booleans
  - Types:
    - `Boolean`
  - `Boolean` values can either be `true` or `false`
