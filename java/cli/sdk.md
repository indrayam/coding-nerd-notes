# sdk

## Useful Commands

```bash
sdk help
sdk help list
sdk help use
sdk install java 23.0.1-tem
sdk install java 21.0.5-tem
sdk ls java | grep -i tem
# Notice the >>> That shows which version is currently in use
# Notice the installed versions called out explicitly
# Temurin       | >>> | 23.0.1       | tem     | installed  | 23.0.1-tem          
#               |     | 21.0.5       | tem     | installed  | 21.0.5-tem
sdk use java 23.0.1-tem
# OR
sdk u java 23.0.1-tem
sdk use java 21.0.5-tem
# OR
sdk u java 21.0.5-tem
```
